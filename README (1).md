# 📰 Way2News Clone

A responsive **Way2News-inspired News Application** built using **HTML, CSS, JavaScript, Node.js, Express.js, and REST APIs**. This project provides users with a clean and mobile-friendly interface to browse the latest news, view live updates, and experience a simplified version of the Way2News platform.

---

## 📌 Project Overview

The Way2News Clone is a news aggregation web application designed to deliver trending and live news articles in an easy-to-read format. The project includes user authentication pages, live news feeds, and a responsive UI optimized for both desktop and mobile devices.

This project was developed for learning purposes to understand:

- Frontend Web Development
- Backend API Development
- RESTful Services
- Responsive Design
- Client-Server Communication

---

## 🚀 Features

- ✅ Responsive User Interface
